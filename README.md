# My Looker Community Visualization

This is a simple Looker Studio community visualization (CVC) hosted on GitHub Pages.

## How to Use

1. Deploy this repo on GitHub (see below).  
2. Enable GitHub Pages (Settings → Pages → Source).  
3. Copy the `viz.js` URL from GitHub Pages.  
4. In Looker Studio:  
   - Go to **Resource → Manage added data sources → Community Visualizations → Add a Community Visualization**  
   - Paste the URL.  
5. Use the visualization in your report.

## Files

- `viz.js`: The Looker Studio visualization code.  
- `index.html`: For testing locally / viewing in browser.  
- `.nojekyll`: Prevent Jekyll from interfering with static file serving.
